# summarize object of class [`sdcProblem-class`](https://sdctools.github.io/sdcTable/reference/sdcProblem-class.md) or [`safeObj-class`](https://sdctools.github.io/sdcTable/reference/safeObj-class.md).

extract and show relevant information stored in object ofs class
[`sdcProblem-class`](https://sdctools.github.io/sdcTable/reference/sdcProblem-class.md)
or
[`safeObj-class`](https://sdctools.github.io/sdcTable/reference/safeObj-class.md).

## Usage

``` r
# S4 method for class 'sdcProblem'
summary(object, ...)
```

## Arguments

- object:

  Objects of either class
  [`sdcProblem-class`](https://sdctools.github.io/sdcTable/reference/sdcProblem-class.md)
  or
  [`safeObj-class`](https://sdctools.github.io/sdcTable/reference/safeObj-class.md).

- ...:

  currently not used.
